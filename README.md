# Blogging Application

A full-stack blogging application built using Spring Boot.

##  Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- REST APIs

##  Features
- Create, update, delete blog posts
- User authentication
- RESTful APIs
- Database integration using Hibernate

##  How to Run
1. Clone the repository
2. Open in Eclipse / STS
3. Run as Spring Boot Application
